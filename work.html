myFlix App (React)

This project is a single-page web app developed using REACT, JavaScript, and Parcel / Node.js build tools. This functioning CRUD app uses a REST API database (hosted on Heroku) to register new users and authenticates them via secure password hashing. Users can browse movies, explore directors/genres, add movies to their list of favorites and update their profile.

  • JWT
